# Security Policy

MornDraft OSS is a local-first public client. The public edition intentionally excludes account login, payment, hosted/private AI providers, AI usage ledger, MCP, telemetry, admin surfaces, hosted links, SMS, and private backend code.

## Reporting a vulnerability

Please use GitHub Security Advisories for private vulnerability reports. Do not open a public issue for suspected vulnerabilities, secrets, auth bypasses, or supply-chain findings.

## Supported versions

The default branch of the public repository is the supported OSS line. Security fixes are shipped through the next public mirror export.
